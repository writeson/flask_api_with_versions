Welcome To Rectangle API

The Rectangle web application is an example framework to try out an XML API using Flask_RestFUL

Most of this is to try out an API along with nested Flask Blueprints to see how well they get along to structure an application.

The API begins at /api and follows this list:

  • HTTP POST /api/v1/{int:merchant_id}/payment - XML payload in body
  • HTTP POST /api/v2/{int:merchant_id}/payment - XML payload in body
